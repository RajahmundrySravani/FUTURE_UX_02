# Mobile App UI for Appointment Booking

## Project Overview
This project is part of Future Interns UI/UX Task 2 (2026).

The goal of this project is to design a mobile-first appointment booking application for a local service business.

## Business Selected
Glow Salon, Rajahmundry

## Problem Statement
Many local businesses manage appointments manually using phone calls, WhatsApp, or registers. This leads to missed bookings, confusion, and poor user experience.

## Solution
This mobile app design provides a simple and efficient way for users to book appointments easily and for businesses to manage their schedule effectively.

## Screens Designed
- Onboarding / Login Screen
- Service Selection Screen
- Staff Selection Screen (optional)
- Date & Time Booking Screen
- Appointment Confirmation Screen

## Key Features
- Clean and modern mobile UI
- Easy booking flow
- Touch-friendly design
- Simple navigation
- Fast and smooth user experience

## User Flow
Login → Select Service → Choose Time → Confirm Booking

## Design Rationale
- The app is designed for salon customers to book appointments easily  
- The flow is kept simple to reduce user confusion  
- Large buttons are used for better mobile usability  
- The layout is clean and minimal for first-time users  
- The design focuses on quick booking with fewer steps
